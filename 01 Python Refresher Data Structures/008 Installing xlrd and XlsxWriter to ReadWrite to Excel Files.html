If you are on Windows, open Command Prompt (CMD), or if you are on Linux or Mac, open Terminal. Then, use the following commands to install xlrd and XlsxWriter packages required for reading and writing to Excel files in the next two lectures.


To install XlsxWriter for writing to Excel files:

pip install XlsxWriter 


To install xlrd for reading Excel files:

pip install xlrd 


If you are on Linux or Mac, please start your Terminal command with sudo
