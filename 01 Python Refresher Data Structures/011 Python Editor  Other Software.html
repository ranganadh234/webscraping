Python

The codes in this course have been tested on Python 3, but it is up to you to adapt them for Python 2.

As for packages required, there is a dedicated lecture for them.


Editor

In this course, PyCharm Community Edition is being used; it is free, and can be downloaded at:

https://www.jetbrains.com/pycharm/download/

However, it is up to you to use any Python editor you prefer.


Operating System

Any. Still, if you face any operating system related issue while installing packages, please send it to the Q&A section of this course.


Internet Connection

Needless to say, web scraping requires Internet connection. However, there are no special requirements. Apparently, faster connection is better for large projects, but in this course, most exercises are not that big.


Instagram Account

This course has a project to log in and scrape images from your Instagram account. While it is generally okay to use your account for this one specific project, it is recommended to create a new account for such testing tasks.


Free Spreadsheets Software

As this course has some lectures about reading and writing to Excel files, you may like to have any free spreadsheet application just to check files. The bottom line is that it is NOT required for the code to work, but if you just want to check files yourself, you can use any free software. This can be any open-source desktop application such as OpenOffice or LibreOffice, or even any free online service such as Google Sheets. Apparently, if you already have MS Excel application, it is okay, but is not required.


If you have any questions, please feel free to send them to the Q&A section of this course.
